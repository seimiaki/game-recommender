# 🎮 NEXTPLAY

### Find your next favorite game.

## 📖 About

NEXTPLAY is a game recommendation platform designed to help players discover their next favorite game.

Users can select genres, search for games, and receive personalized recommendations through an interactive and modern gaming interface.

## ✨ Current Features

- 🎮 Interactive game recommendation interface
- 🏷️ Multi-category game selection
- 🪟 Animated recommendation modal
- ❤️ Favorites system (in progress)
- 🎧 Button sound effects
- 🌌 Animated glossy background
- ✨ Smooth UI animations
